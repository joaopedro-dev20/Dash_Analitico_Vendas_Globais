Dashboard Analítico de Vendas Globais

Este repositório contém um dashboard desenvolvido em Power BI com foco na análise de vendas globais. O projeto foi criado para praticar habilidades de modelagem, visualização e interpretação de dados, utilizando um conjunto de dados de vendas internacionais.

Descrição do Dashboard

O relatório apresenta uma visão geral das vendas por diferentes perspectivas, permitindo explorar resultados de forma interativa. Entre as principais análises estão:

1. Média de Desconto por Produto

Gráfico de barras exibindo a média de desconto aplicada em cada subcategoria de produto. Permite identificar produtos com maior ou menor política de desconto.

2. Total de Vendas por Categoria

Gráfico de pizza destacando a participação de cada categoria de produto no total das vendas globais. Inclui percentuais e valores totais.

3. Valor Total Vendido

Indicador numérico mostrando o montante total de vendas consolidadas no período analisado.

4. Média de Vendas por País

Mapa geográfico que apresenta a média de vendas por país, permitindo identificar regiões de maior concentração de receita.

5. Vendas por País com Detalhamento de Prioridade

Gráfico de barras horizontais exibindo as vendas por país, segmentadas por nível de prioridade (alto, médio, baixo e crítico).

6. Filtros de Análise

O dashboard inclui filtros que permitem ao usuário personalizar a visualização:

Ano (2011–2014)
Segmento (Consumidor, Corporativo, Home Office)
País

Esses filtros possibilitam análises direcionadas e comparações entre períodos, regiões e segmentos específicos.

Ferramentas Utilizadas

Power BI Desktop
Power Query para transformação de dados
Modelagem relacional simples seguindo boas práticas
Mapas e gráficos nativos do Power BI
